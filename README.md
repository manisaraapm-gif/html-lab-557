สิ่งที่ได้จากการทำงานนี้คือทำให้เข้าใจการใช้งาน HTML มากขึ้น และทำให้สามารถใช้โครงสร้างของ HTML แบบ semantic ได้ถูกต้อง ทำให้เว็บมีความเป็นระเบียบอ่านง่ายตามวัตถุประสงค์ และยังได้ลองใช้ tags ทำให้รู้ว่า tag แต่ละชนิดใช้ทำอะไรได้บ้าง โดยไม่มีการใช้ CSS ในการช่วยตกแต่ง จึงรู้ได้อีกว่าการเขียนเว็บที่ดีและอ่านง่ายนั้นอยู่ที่การเขียนโครงสร้างให้ถูกต้องและเขียนเนื้อหาให้เข้าใจง่าย เป็นคัวแปรสำคัญที่จะช่วยทำให้เว็บอ่านได้ง่ายมากขึ้น
